# 大学生个人主页

这是一个为大学生设计的个人主页网站，旨在清晰展示在校身份、学习成果和实践能力，兼顾「学生特质」和「潜力展示」。

## 文件结构

```
├── index.html          # 主页HTML文件
├── style.css           # 样式文件
├── script.js           # JavaScript交互文件
└── README.md           # 项目说明文件
```

## 如何发布到GitHub

由于系统中没有安装git，您需要手动创建GitHub仓库并上传文件：

### 步骤1：创建GitHub仓库
1. 登录您的GitHub账号
2. 点击右上角的「+」按钮，选择「New repository」
3. 填写仓库名称（例如：personal-homepage）
4. 选择「Public」或「Private」
5. 勾选「Initialize this repository with a README」
6. 点击「Create repository」

### 步骤2：上传文件
1. 进入创建好的仓库
2. 点击「Add file」按钮，选择「Upload files」
3. 拖拽或选择本地的 `index.html`、`style.css` 和 `script.js` 文件
4. 填写提交信息（例如：「Add personal homepage files」）
5. 点击「Commit changes」

### 步骤3：配置GitHub Pages
1. 在仓库页面，点击「Settings」选项卡
2. 滚动到「GitHub Pages」部分
3. 在「Source」下拉菜单中，选择「main」分支
4. 点击「Save」
5. 稍等几分钟，GitHub Pages会生成您的网站

### 步骤4：访问网站
1. 在「GitHub Pages」部分，您会看到网站的访问链接（例如：https://yourusername.github.io/personal-homepage）
2. 点击链接即可访问您的个人主页

## 网站内容说明

### 核心必备信息
- 基础身份标识：姓名、头像、核心标签
- 校园背景：学校、学院、专业、入学/毕业时间、学历层次
- 核心学习能力：专业技能、核心课程
- 便捷联系方式：邮箱、GitHub、微信

### 进阶加分信息
- 校园实践/经历：学生工作、实习经历、校园项目
- 竞赛/获奖成果：学科竞赛、奖学金/荣誉、证书
- 学习/项目成果：课程设计、科研相关、学习计划

### 个性化补充信息
- 兴趣爱好
- 个人理念/学习感悟
- 额外链接

## 响应式设计

网站采用响应式设计，适配不同设备屏幕：
- 桌面端：多列布局
- 平板端：适当调整布局
- 移动端：单列布局

## 自定义修改

您可以根据自己的实际情况修改以下内容：
1. 个人信息（姓名、头像、标签等）
2. 校园背景信息
3. 技能和课程信息
4. 实践经历和获奖成果
5. 兴趣爱好和个人理念
